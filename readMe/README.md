# 🌐 Webサイト開発ポートフォリオ（Html / Javascript / Css）

## 📘 概要
Html / Javascript / Css を用いて開発した Webサイトです。
障害を持つ人とのコミュニケーションを高めるため、手話学習用の動画検索サイトを作成しました。
検索・ブックマークなどの処理を実装して学習目的で作成しました。
